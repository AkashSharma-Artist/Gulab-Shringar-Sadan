# Gulab Shringar Sadan — Website
 
A single-page website for **Gulab Shringar Sadan**, a shringar (bridal accessories) and gift shop in Utai, Chhattisgarh. Built with plain HTML, CSS, and JavaScript — no framework or build step required.

## Features
 
- Responsive layout (desktop + mobile nav with slide-out menu)
- Sections: Home / Hero, Our Story (About), Shringar collection, Gifts collection, FAQ, Visit Us / Contact
- Contact form that hands off directly to WhatsApp
- Smooth scroll-reveal animations
- Back-to-top button
- Fully static — works by just opening `index.html`, no server needed
